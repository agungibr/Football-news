# Football News
in this project I created a website that is specialized in news about football sports using PHP and MySQL.

## Built With
* PHP
* CSS
* MySQL

## Features
* Index Page
* Login
* Main Page
* Add News
* Add Category
* Edit News
* Edit Category
* Delete News
* Delete Category
* Search
* Subscibe
* Logout
